# AI Image Generator SaaS

This is a full-stack AI-based SaaS application that allows users to generate images from text prompts using the ClipDrop API.

The platform includes user authentication, a credit-based system, and online payment integration for purchasing additional credits.

---

## Overview

Users can register and log in securely. Each new user is given **5 free credits**. Every time a user generates an image, one credit is used. If credits run out, users can purchase more through Razorpay using different plans.

The application is built with a scalable backend using **Node.js, Express.js, and MongoDB**, and a responsive frontend using **React**.

---

## Features

- Generate images from text prompts using AI  
- User registration and login  
- Credit-based usage model (5 free credits per user)  
- Razorpay payment integration  
- RESTful API architecture  
- Responsive frontend  

---

## Tech Stack

**Frontend:** React  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**API:** ClipDrop  
**Payments:** Razorpay  

---

## How It Works

1. User signs up and logs in  
2. Gets 5 free credits  
3. Enters a text prompt  
4. Image is generated using credits  
5. User can buy more credits if needed
